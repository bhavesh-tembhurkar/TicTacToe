Tic-Tac-Toe – iOS App

A simple two-player Tic-Tac-Toe game built for iOS using SwiftUI.
The app demonstrates clean UI design, smooth state management, and a well-structured MVVM architecture.

---

## Features
- 3×3 interactive game board  
- Two-player gameplay on the same device  
- Current player indicator (X or O)  
- Win detection with highlighted tiles  
- Draw detection  
- Restart game option  
- Smooth animations and clean UI  

---

## Architecture

The project follows MVVM (Model–View–ViewModel):

- **Model** – Represents cell states and player types  
- **ViewModel** – Contains game logic, win/draw detection, and board updates  
- **View** – SwiftUI views that render the board and respond to ViewModel changes  

---

## Tech Stack
- Swift  
- SwiftUI  
- MVVM  
- Xcode  

---

## How to Run

1. Clone the repository  
2. Open the project in Xcode  
3. Select an iOS simulator or a physical iPhone device  
4. Press **Run (⌘ + R)**
